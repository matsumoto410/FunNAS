Fun NAS Game

This game is developed on NEBULAS chain. Nebulas is a next generation public blockchain, aiming for a continuously improving ecosystem. Based on its blockchain valuation mechanism, Nebulas proposes future-oriented incentive and consensus systems, and the ability to self-evolve without forking.

Game Rules：
1. This is one collection game on the cells which make up the word "NAS".
2. For all the initial cells, they are free and have no owner, you can choose any cells you like.
3. For the cells you own, you can set the selling price, without paying any fees.
4. For the cells you own, you can set the background color you like, without paying any fees.
5. There will be one 5% fees when you buy the cells from other owners.
